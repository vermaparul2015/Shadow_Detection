////////////////////////////////////// SHADOW DETECTION //////////////////////////////////////////////////////
***************************************************************************************************************


################################# MINIMUM SYSTEM REQUIREMENTS #################################################



1. Windows Operating System
2. RAM: 4 GB

3. MATLAB version 2013Ra or higher with Image Processing Toolbox


#################################### MATLAB INSTALLATION AND STARTUP STEPS ####################################




1. Download the latest version of MATLAB from www.mathworks.com.


2. Start the installer, enter the activation key and install the required packages.


3. CLick open the MATLAB icon to start the environment.



#################################### CONTENTS OF SHADOW DETECTION FOLDER #######################################



1. Shadow_Dection.m  %contents the end results of all the applied techniques

2. Shadow Dection.docx  %project report

3. Modified_Triclass_Thresholding.m  %code for modified triclass thresholding

4. HSV_Thresholding.m  %code for HSV thresholding

5. K_Means.m  %code for k means based shadow detection technique

6. Fuzzy_C_Means.m  %code for fuzzy c means based shadow detection

7. A set of images in .jpg format.



############################################# EXECUTION #########################################################



1. Click open MATLAB icon.

2. Open the ShadowDetection folder in MATLAB.

3. From the Current Folder list select and open the Shadow_Dection.m

4. Click the Editor tab and press the Run button to execute the file.

5. Shadow_Dection_All_Methods GUI will open on screen.

6. Click select Image button. A window will open showing the content of ShadowDection folder. Select any image
   
to load it.

7. Click buttons Modi_Tri_Class_Thresholding, HSV_Thresholding, K_Means and Fuzzy_C_Means one at a time and wait 
   
to see the corresponding results.



################################################# EXTRAS ########################################################



Alternatively, to view results on a larger window, each method can be executed separately from there respective

files.
